OMNI Extraction Instructions

Here are some general guidelines for extracting data via OMNI to get accurate results via Excel details concatenation:

  1. Ensure Each Line is Drawn: Verify that all lines in your data are properly drawn and visible. This ensures that the data extraction process can accurately capture all necessary information.
  2. Capture Pen Marks Amounts: Make sure that the amounts marked with a pen are correctly captured against each transaction. This is crucial for maintaining the accuracy of the extracted data.
  3. Check for Completeness: Ensure that all required fields are filled out and that no data is missing. Incomplete data can lead to errors in the extraction process.
  4. Verify Data Accuracy: Double-check the extracted data for any discrepancies or errors. This includes verifying that the amounts, dates, and other details match the original source.
  5. Avoid Data Merging: Make sure that no merged cells in extracted data.

Concatenation Instructions

General guidelines to ensure accurate results when concatenating details in Excel:

  1. Check Transaction Values: Ensure that all transactions have values in both the debit and credit columns. Missing values can lead to inaccurate concatenation results.
  2. Extract Details Properly: Make sure that the details are properly extracted and that there is no merged data. Merged cells can cause issues with data manipulation and concatenation.
  3. Use Headers: Always give headers to your data. This helps in identifying the columns correctly and makes the data more readable.
  4. Validate Date Column:Give proper formatting to date column to avoid any wrong date issue after concatenation.
  5. Validate Results: After concatenation, validate the results to ensure that the data has been combined correctly and that no information is lost or misrepresented.
